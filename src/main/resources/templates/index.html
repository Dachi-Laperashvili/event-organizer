Event Organizer

Event organizer application, which lets some group of people organize an event.

Key features:

Event Creation

Users can create and set up events.This feature allows users to efficiently plan their events.

User Collaboration

Users can invite other participants to join the event and collaborate with them.

Task Management

The application offers task management,allowing users to create and track tasks.

Payment details

Application tracks and manages payment details associated with tasks and view payment summaries.
